PRD — 학부연구생을 위한 논문 선수 지식 분석 SaaS
1. 문서 개요
1.1 제품명

PaperPrep
부제: 논문을 읽기 전에, 먼저 알아야 할 것을 구조화해주는 학부연구생용 연구 학습 SaaS

1.2 제품 한 줄 설명

사용자가 논문 PDF를 업로드하면, AI가 해당 논문을 이해하기 위해 필요한 선수 지식(prerequisite knowledge) 을 추출하고, 이를 트리 형태의 학습 로드맵으로 제공하며, 논문 읽기 과정에서 용어 설명·맥락 기반 Q&A·개인 지식 프로필 관리까지 지원하는 학습 보조 서비스.

1.3 핵심 문제

학부연구생이나 연구 초심자는 논문 자체보다도 그 논문이 전제하는 배경지식 때문에 논문 읽기에 실패하는 경우가 많다.
현재의 일반적 AI 요약 도구는 논문의 내용을 요약해줄 수는 있지만, 다음을 체계적으로 해결하지 못한다.

이 논문을 이해하려면 무엇을 먼저 알아야 하는지
그 배경지식들 사이의 선후관계가 무엇인지
사용자가 이미 아는 것과 모르는 것을 구분해 개인화된 학습 경로를 만드는 방법
논문을 읽는 중 등장하는 개념을 맥락 안에서 즉시 설명하는 기능
1.4 제품 목표

이 제품의 1차 목표는 “논문 요약”이 아니라, 논문 이해를 위한 선수 지식 구조화와 학습 지원이다.

2. 제품 목표 및 비목표
2.1 목표
사용자가 논문을 업로드하면 텍스트를 안정적으로 추출·분석할 수 있어야 한다.
논문 이해에 필요한 선수 지식을 트리 형태로 시각화해야 한다.
각 지식 노드마다 난이도별 요약과 추천 학습 링크를 제공해야 한다.
사용자가 논문을 읽는 중 특정 문장·용어·수식에 대해 맥락 기반 질문을 할 수 있어야 한다.
사용자의 학습 이력과 이해 수준을 반영해 지식 프로필을 관리해야 한다.
2.2 비목표

4. 핵심 가치 제안

본 서비스는 단순 논문 요약기가 아니라 다음을 제공한다.

Prerequisite-first 학습 경험
논문 내용을 바로 요약하는 대신, 이해를 위한 선행 개념을 먼저 구조화한다.
Knowledge graph가 아닌 학습 지향 지식 트리
논문 관련 개념을 단순 나열하지 않고, “무엇을 먼저 이해해야 하는가”를 기준으로 정렬한다.
Reading-in-context 지원
논문을 읽는 과정에서 등장하는 개념을 현재 문맥에 맞게 설명한다.
개인화된 학습 축적
사용자가 이미 익힌 개념과 부족한 개념을 기록하여 이후 로드맵 생성에 반영한다.

7.2 기능 2 — 선수 지식 로드맵 생성
7.2.1 목적

논문 이해를 위해 필요한 배경지식을 계층 구조로 추출하고, 사용자가 먼저 공부할 수 있도록 학습 로드맵을 제공한다.

7.2.2 사용자 스토리
사용자는 논문을 읽기 전에 어떤 개념을 먼저 알아야 하는지 알고 싶다.
사용자는 개념 간 선후관계를 시각적으로 보고 싶다.
사용자는 쉬운 설명부터 깊은 설명까지 난이도별로 보고 싶다.
사용자는 각 개념에 대해 추가 학습 링크를 받고 싶다.

논문 구조 분석기

11.2 출력 형식 강제

모든 핵심 분석은 자유 텍스트가 아니라 JSON 스키마 기반 출력을 강제한다.

UI 렌더링 안정성
후처리 용이성
버전 관리 가능
재분석 및 캐싱 가능
11.3 프롬프트 설계 원칙
“이 논문의 요약”보다 “이 논문을 이해하기 위해 필요한 선수 지식”에 집중
개념 수는 과도하게 많지 않게 제한
학부연구생 수준을 기준으로 설명
개념 간 선후관계가 없는 평면 나열 금지

20. 최종 제품 정의

PaperPrep은 논문 내용을 단순 요약하는 서비스가 아니라,
논문을 이해하기 전에 먼저 알아야 할 배경지식을 구조화하고, 읽는 과정에서 맥락형 설명을 제공하며, 장기적으로 사용자의 연구 지식 프로필을 축적하는 학습 인프라 SaaS이다.

핵심 차별점은 다음 한 문장으로 요약된다.

“이 논문이 무슨 말인지 설명하는 것”이 아니라,
“이 논문을 이해하려면 무엇을 먼저 배워야 하는지”를 개인화해서 보여주는 것.
